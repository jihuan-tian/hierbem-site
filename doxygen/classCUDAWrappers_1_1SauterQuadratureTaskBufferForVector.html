template<int dim, int spacedim, typename CUDARangeNumberType , typename CUDAKernelNumberType >
unsigned int* CUDAWrappers::SauterQuadratureTaskBufferForVector< dim, spacedim, CUDARangeNumberType, CUDAKernelNumberType >::starting_task_indices_for_dof_pairs_cpu

Store the list of starting task indices for each pair of DoFs.

Note
For each pair of DoFs \(i\) and \(j\), the total number of associated quadrature tasks depends on the number of cells in the support sets of \(i\) and \(j\). To compute a particular element in the row or column vector of the rank- \(k\) matrix, we need to accumulate the quadrature results from all tasks associated with this vector element. Therefore, we need this array to consecutively store the starting task index for each pair of DoFs. The length of this array is vector_size+1, with the last element being the ending task index for the last pair of DoFs.

The data are on the host.
