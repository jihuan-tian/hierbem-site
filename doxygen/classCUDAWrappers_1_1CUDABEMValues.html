CUDATable<4, RangeNumberType> CUDAWrappers::CUDABEMValues< dim, spacedim, RangeNumberType >::kx_mapping_shape_grad_matrix_table_for_common_edge

Data table of mapping shape function's gradient values for \(K_x\) in the common edge case. It has 5 dimensions:

  1. Mapping index, which is mapping order minus 1. It is used to get the corresponding pointer to the MappingInfo object in BEMValues::mappings.
  2. \(k_3\) term index: size=6
  3. Quadrature point index: size=number of quadrature points
  4. Gradient matrix element index in column major. The total number of elements is (number of shape functions in the highest order mapping object)*dim.
