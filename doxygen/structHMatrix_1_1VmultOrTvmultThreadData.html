Detailed Description

template<int spacedim, typename Number = double>
struct HMatrix< spacedim, Number >::VmultOrTvmultThreadData

Data on a single thread to be used in a parallel (transposed) \(\mathcal{H}\)-matrix/vector multiplication.

The index ranges used in this class are half-closed and half-open, which is different from the closed interval adopted in sequence partition.

template<int spacedim, typename Number = double>
std::array<types::global_dof_index, 2> HMatrix< spacedim, Number >::VmultOrTvmultThreadData::local_tvmult_result_index_range

The index range of the local Tvmult result vector.

It is a continuous index range which covers the column indices of all leaf \(\mathcal{H}\)-matrix blocks assigned to the thread. This is \(J(q)\) in Definition 2.11 in (Bebendorf 2008).

References
Bebendorf, Mario. 2008. Hierarchical Matrices: A Means to Efficiently Solve Elliptic Boundary Value Problems, page 56. Lecture Notes in Computational Science and Engineering. Berlin, Heidelberg: Springer.

If the Z curve is used for traversing the leaf set, this index range is not same as the union of all row index sets from the leaf \(\mathcal{H}\)-matrix blocks, because they may not be contiguous. When the Hilbert curve is used, their union is the same as the index range said above.

Note
This index range is usually different from VmultOrTvmultThreadData::tvmult_result_index_range.
