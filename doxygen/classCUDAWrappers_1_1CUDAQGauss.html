template<int dim>
CUDATable<2, double> CUDAWrappers::CUDAQGauss< dim >::points
private

Table of quadrature points.

  • The first dimension is the quadrature point index.
  • The second dimension is coordinate index.

Since CUDATable adopts C style indexing, this memory configuration is consistent with the quadrature point data held within QGauss, which is a vector of Point objects.
