Detailed Description

template<int spacedim, typename Number = double>
class HBlockMatrixSkewSymm< spacedim, Number >

Class for the skew symmetric block \(\mathcal{H}\)-matrix.

The block \(\mathcal{H}\)-matrix has the structure

\[ \begin{pmatrix} M_{11} & M_{12} \\ (M_{21}) & M_{22} \end{pmatrix}, \]

where \(M_{11}\) and \(M_{22}\) are symmetric and positive definite, the matrix block \(M_{21}=-M_{12}^T\) is not stored.
