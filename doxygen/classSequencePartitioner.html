Detailed Description

template<typename IntervalCostFunc>
class SequencePartitioner< IntervalCostFunc >

Partition the sequence with index range [0,n-1] into p partitions such that the maximum cost over all partitions is minimized.

The algorithm have the memory complexity of \(O(np)\) and the time complexity of \(O(p(n-p))\).

The cost function type is declared as template parameter to allow inlining it by the compiler to achieve the maximum performance.

NOTE: For large sequence the code compiled by Clang 12 with -O3/-Ofast optimization can be ~50% faster than GCC 9.4 with the same flags.

References
Olstad and Manne (1995), "Efficient Partitioning of Sequences" (Fig. 1)
